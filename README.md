# Gulp-starter

### Dev tasks
* **Jade** (HTML)
* **SASS & Autoprefixer** (CSS)
* **Browserify & JSHint** (JS)
* **Browser-sync** (Local server with livereload)

Launch server
```
$ gulp
```
or
```
$ gulp serve
```

Enable watch for generating files
```
$ gulp dev-watch
```

### Build tasks
* **Minify** (HTML/CSS/JS)
* **Deploy** (gh-pages)
```
  $ gulp build
```
